# Учёт транспорта организации

## О проекте
Данный проект разработан по дисциплине **«Технология разработки программного обеспечения»**.

Тема проекта: **«Учёт транспорта организации»**.

Система предназначена для автоматизации работы со служебным транспортом в организации.  
Пользователи системы могут оформлять заявки на использование транспорта, а ответственный сотрудник — распределять транспортные средства, назначать водителей и фиксировать выполненные поездки.

## Цель проекта
Создание информационной системы, которая позволяет упростить процесс учёта служебного транспорта, оформления заявок и контроля поездок.

## Основные задачи проекта
- анализ предметной области;
- формирование требований к системе;
- разработка модели предметной области;
- построение UML-диаграмм и структурных схем;
- проектирование архитектуры системы;
- разработка алгоритмов работы ключевых сценариев;
- реализация демонстрационного программного модуля.

## Основные функции системы
- регистрация и хранение данных о пользователях;
- создание заявок на использование транспорта;
- просмотр и обработка заявок;
- распределение транспортных средств;
- назначение водителя;
- фиксация информации о поездке;
- хранение истории заявок и поездок.

## Используемые технологии
- **Язык программирования:** Python
- **Тип приложения:** информационная система
- **Среда разработки:** может быть использована Visual Studio Code, PyCharm или другая IDE
- **Хранение данных:** база данных

## Ключевые сценарии работы
1. Сотрудник создаёт заявку на использование транспорта.  
2. Ответственный сотрудник рассматривает заявку.  
3. Система позволяет назначить транспорт и водителя.  
4. После выполнения поездки фиксируются её результаты.  
5. Данные сохраняются для дальнейшего учёта и контроля.

## Ожидаемый результат
Результатом проекта является разработка учебной информационной системы для учёта транспорта организации, а также комплект проектной документации, включающий модель предметной области, требования, диаграммы, архитектурные решения и демонстрационный программный код.
